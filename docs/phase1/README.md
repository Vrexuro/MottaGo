# Phase 1 - Discover

## Tujuan

Fase Discover bertujuan untuk menganalisis dan mengekstrak seluruh kebutuhan sistem berdasarkan Product Requirements Document (PRD).

Pada fase ini tidak dilakukan desain, implementasi, maupun pengambilan keputusan teknis.

## Deliverables

* Functional Requirements
* Non-Functional Requirements
* Stakeholder Needs Mapping
* Requirements Traceability Matrix
* Data Requirements
* Workflow Requirements
* Dependency Analysis

## Rules

* PRD merupakan sumber kebenaran utama (Single Source of Truth).
* Tidak diperbolehkan menambahkan fitur yang tidak terdapat dalam PRD.
* Tidak diperbolehkan membuat keputusan desain atau implementasi.
* Semua requirement harus dapat ditelusuri kembali ke PRD.
