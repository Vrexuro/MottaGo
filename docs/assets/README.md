# Assets

## Tujuan

Folder ini digunakan untuk menyimpan seluruh aset pendukung dokumentasi proyek MottaGo.

Aset yang disimpan di sini bukan merupakan source code, melainkan file visual, diagram, desain, dan artefak dokumentasi yang digunakan selama proses analisis, desain, dan pengembangan sistem.

## Struktur Folder

### bpmn/

Digunakan untuk menyimpan:

* BPMN Diagram
* Workflow Diagram
* Business Process Diagram

### erd/

Digunakan untuk menyimpan:

* ERD (Entity Relationship Diagram)
* Logical Data Model
* Physical Data Model
* Database Relationship Diagram

### wireframes/

Digunakan untuk menyimpan:

* Low Fidelity Wireframe
* Mid Fidelity Wireframe
* User Flow Sketches

### mockups/

Digunakan untuk menyimpan:

* High Fidelity UI Design
* Dashboard Mockup
* Screen Mockup
* Final UI Screens

### presentations/

Digunakan untuk menyimpan:

* Presentasi Progress
* Presentasi UTS/UAS
* Presentasi Final Project

## Rules

* Gunakan nama file yang deskriptif dan konsisten.
* Hindari penggunaan nama file seperti:

  * image1.png
  * final_final.png
  * terbaru.png

Contoh yang baik:

* BPMN_Waste_Collection_v1.png
* ERD_MottaGo_v2.drawio
* Dashboard_Manager_Mockup.png

## Versioning

Jika terdapat revisi besar, gunakan format:

filename_v1.ext

filename_v2.ext

filename_v3.ext

## Ownership

Seluruh anggota tim dapat menambahkan aset sesuai kebutuhan proyek.

Perubahan penting yang memengaruhi requirement, workflow, atau desain utama harus dicatat pada DECISION_LOG.md.
